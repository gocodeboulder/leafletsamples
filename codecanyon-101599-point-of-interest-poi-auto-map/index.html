Google Maps API Version 3 and Google Places API

POI Auto Maps Version 5 has been re-written to work with Google Maps API V3 which is the currently supported version in unison with the Google Places API. You can now retriev a large number of POI items.

Database Driven Layers

You can add your own markers from your own database. Each marker contains your own title and HTML for the popup info window. Each database group has a default marker icon, or you can define an individual icon per poi point. In this example, the Parking and Playground markers are database driven with a standard marker for each group. The "My Photographs"and "Property For Sale marker each have their own unique icon. All other layers are generated from Google Places API.
